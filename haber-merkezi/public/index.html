Aiaddin Haber Merkezi
API + ayrı Postgres

Çoklu haber sitesi için merkezi kayıt ve API

Bu alan adı (aiaddin.net) Haber Merkezi ürününün vitrin ve yönetim girişidir. Asıl REST API Railway üzerinde çalışır; veritabanı Yekpare (Goalgo) veritabanından bağımsızdır.

Railway

haber-merkezi Docker imajı: migrasyon + API. Sağlık ucu /healthz. Operatör uçları /v1/operator/… (Bearer OPERATOR_API_KEYS).

Yekpare.net

Portal ve mevcut haber vitrinleri ayrı projede kalır. Sonraki fazda Yekpare sunucusu, panelden yaptığınız işlemleri bu API’ye güvenli proxy ile iletecek; tarayıcıya gizli anahtar verilmez.

Vercel

Bu statik sayfa Vercel’de yayınlanabilir. Domain olarak aiaddin.net bağlayın; kök dizin olarak repo içinde haber-merkezi seçin.

Mevcut haber siteleri

Örn. ankarasehirgazetesi.com hâlâ Yekpare ön yüz derlemesine gider; Haber Merkezi deploy’u onları tek başına taşımaz. Taşıma ve çift kayıt önleme için yol haritasına bakın.

Sizin sıradaki adımlar

  1. Vercel: Proje → Root haber-merkezi → Production domain: aiaddin.net (DNS’te CNAME/A yönergelerini uygulayın).
  2. Railway: Ayrı servis + ayrı Postgres; değişkenler DATABASE_URL, OPERATOR_API_KEYS, NODE_ENV=production.
  3. Yekpare (Goalgo): Aynı repo; deploy sonrası özel alan siteleri önceki gibi çalışır. Haber Merkezi entegrasyonu ayrı commit/faz ile gelir.
  4. Detaylı faz listesi: repoda haber-merkezi/docs/AIADDIN_YOL_HARITASI.md
